### Toby-seed-note 규칙
0. 해당 주차별 공부 기간은 Issue로 등록하여 공지한다.
1. 주차별로 정해진 범위까지 공부한 내용 및 코드를 블로그 및 노션에 기록한다.
2. 기록에 url을 발표 격주로 진행하는 해당 주차 README.md 파일을 수정하여 추가한다.
3. `[이름] n주차 발표 업로드` 형식으로 커밋을 한다.
